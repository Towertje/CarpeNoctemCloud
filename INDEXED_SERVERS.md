# Indexed servers

This document outlines the servers the website indexes and how these servers where found. If your
server is in here, and do not want it to be indexed, please create an issue so it can be removed. If
you want to add your server to the indexing, please make another issue stating that you wish for it
to be indexed.

## nmblookup

The command `nmblookup -S '*'` was used to scan the University of Twente network to find open
drives. These where the drives that where found:

* univac.student.utwente.nl
* stroopwafel.student.utwente.nl
* spitfire.student.utwente.nl
* campuslaan53.student.utwente.nl
