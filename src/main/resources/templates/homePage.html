Welcome to Carpe Noctem Cloud

Carpe Noctem Cloud (CNC) is a rudimentary clone of UT Cloud. It indexes various file servers located around campus and allows searching and downloading from one place. After finding a file one wishes to download, an HTML file will be downloaded which contains a redirect to the file itself. Opening the HTML file will prompt to download and voil√†! you have successfully used CNC.

Important notes
Currently, when downloading, it will download an archive file (.zip). This is to prevent browsers from adding .download to the end of the files and breaking the ability to click the URL files.
When following a redirect, it might ask for authentication. Leaving the password empty and pressing next should do the trick.
